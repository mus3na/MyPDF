# Introduction
Welcome to MyPDF — a professional yet lightweight PDF reader and editor built for Windows. This manual walks you through every feature of the application step by step, from opening your first document to advanced tasks like redaction and e-signing.


# What Can MyPDF Do?
MyPDF combines the essential features of a commercial PDF editor into a single clean interface:
•	Open and read PDF files, including password-protected documents.

•	Organize pages — drag to reorder, split a document, insert pages from another PDF, or delete pages.

•	Edit text directly on the page.

•	Place signatures and stamps (E-Sign) with drag-and-resize overlays.

•	Add diagonal watermarks with configurable text, angle, size, and transparency.

•	Black-out (redact) sensitive information permanently.

•	Print documents to any installed printer.

•	Full undo and redo history for all edits.


# System Requirements
•	Operating System: Windows 10 or later (64-bit).

•	RAM: 4 GB minimum, 8 GB recommended for large documents.

•	Disk Space: 200 MB for installation.

•	Display: 1280 × 720 minimum resolution.

# Reporting Bugs
When reporting a bug, please include:
•	Your Windows version (Windows 10, 11, etc.).

•	The steps that produce the bug (e.g., "Open file X, click Redact, drag across line 3").

•	The exact error message from the status bar or any popup dialog.

•	If possible, a screenshot of the problem.

•	If the bug involves a specific PDF, attach a sample file (or indicate if the file is confidential).
